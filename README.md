# Kaizen Impact Tracker

A lean continuous improvement tracker for managing Kaizen initiatives. It provides a Kanban board to visualize workflow, track before/after metrics, calculate ROI, and display impact dashboards. This project explores building a practical tool to quantify the results of incremental improvement processes.

## Key Features
*   **Kanban Board:** Visualize and manage Kaizen initiatives through customizable stages.
*   **Metrics & ROI Tracking:** Log before/after states and automatically calculate return on investment.
*   **Impact Dashboards:** View consolidated reports and charts on improvement impact.
*   **Initiative Management:** Full CRUD operations for improvement projects and ideas.

## Tech Stack
*   **Frontend:** React
*   **Backend:** FastAPI (Python)
*   **Database:** SQLite

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/kaizen-impact-tracker.git`
2.  Install backend dependencies: `pip install -r requirements.txt`
3.  Install frontend dependencies: `npm install`
4.  Run the development servers (see individual `/backend` and `/frontend` READMEs for details).
